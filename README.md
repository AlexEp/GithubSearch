# GithubSearch


## And don't forget :
1. Run "npm i" in the GithubSearch.WEB project
```bash
 npm i
```
2. (option) Run "docker-compose up -d" in the docker-compose.yml folder to create a mongoDB instance
```bash
 docker-compose up -d
```


3. Config the appsettings.json in the GithubSearch.API project


Thanks!
